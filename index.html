Welcome to Magpie

The Oncoblocks Genomic Data Warehouse Project for Google Summer of Code 2015.

About the project

Our goal is to create a specification and refernece data warehouse solution for processed genomic data, helping you consolidate, annotate, and share your data easier.

Modular and Scalable
Create your own data warehouse using the Oncoblocks data model, or bootstrap one or more RDBMS or NoSQL data sources to our demo implementation.
RESTful API
Access your data through a flexible and performant REST API, with customizable features you can tailor to your organization's needs.
Free and Open-Source
Magpie is built with free and open source technologies, and can be integrated with other tools from the Oncoblocks platform.
